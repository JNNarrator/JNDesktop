# JNTool 开发计划

## 项目概述
- **项目名称**: JNTool
- **目标平台**: macOS
- **框架**: Flutter
- **组织标识**: com.jntool
- **UI风格**: 现代可爱风格（柔和配色、圆角、毛玻璃效果、微动效、Emoji点缀）

## 开发阶段

### 阶段一：项目初始化
1. 使用 Flutter 创建 macOS 项目
2. 配置基础项目结构
3. 设置开发依赖

### 阶段二：核心架构搭建
1. 创建应用入口和主界面
2. 实现侧边栏导航组件
3. 实现工具卡片组件
4. 建立工具注册机制

### 阶段三：UI 美化
1. 实现毛玻璃效果
2. 添加柔和渐变背景
3. 配置圆角和阴影
4. 添加微动效和过渡动画

### 阶段四：测试与验证
1. 运行应用验证基础功能
2. 测试 UI 响应性
3. 打包验证

## 文件结构
```
lib/
├── main.dart                 # 应用入口
├── app.dart                  # 应用主组件
├── models/
│   └── tool_model.dart       # 工具数据模型
├── widgets/
│   ├── sidebar.dart          # 侧边栏导航
│   ├── tool_card.dart        # 工具卡片
│   └── glass_container.dart  # 毛玻璃容器
├── screens/
│   └── home_screen.dart      # 主界面
└── utils/
    └── constants.dart        # 常量配置
```

## 设计规范
- **主色调**: 粉蓝紫渐变
- **字体**: 系统默认 + SF Pro
- **圆角**: 12-20px
- **阴影**: 柔和扩散阴影
- **动画**: 200-300ms 缓动

## 后续扩展
- 工具功能模块将逐步添加
- 每个工具作为独立模块注册
- 支持工具发现和搜索
